## 🚀 Projects Overview

### 1. Data Structures and Creating Functions in Python
*   **Directory:** `Data Structures and Creating Functions in Python/`
*   **Description:** Covers foundational Python programming concepts.
*   **Key Topics:** Python lists, dictionaries, tuples, sets, custom function definitions, and control flow logic.

### 2. Exploratory Data Analysis (EDA) with the Sales Dataset
*   **Directory:** `Exploratory Data Analysis (EDA) with the Sales Dataset.../`
*   **Dataset:** `online_sales_retailer.xlsx` (located in the `Source Data` subfolder).
*   **Description:** Analyzing retail sales trends, customer behavior, and product performance.
*   **Libraries Used:** `pandas`, `matplotlib`, `seaborn`

### 3. Exploring and Analyzing Data with Pandas
*   **Directory:** `Exploring and Analyzing Data with Pandas/`
*   **Description:** Deep dive into data manipulation techniques.
*   **Key Topics:** Data cleaning, handling missing values, filtering rows, grouping, and aggregating datasets.

### 4. Predicting House Price with Multiple Linear Regression
*   **Directory:** `Predicting House Price with Multiple Linear Regression/`
*   **Description:** A predictive modeling project estimating housing prices based on multiple continuous features.
*   **Key Topics:** Feature selection, model training, evaluation metrics (R², RMSE), and assumption testing.

## 🛠️ Requirements & Installation

Ensure you have Python 3.x installed, then install the necessary dependencies via terminal:

```bash
pip install pandas openpyxl matplotlib seaborn scikit-learn
```

> **Note:** The `openpyxl` engine is required by Pandas to read the `.xlsx` Excel source data file.

## 💻 How to Run

Navigate to any specific project directory and run the script:

```bash
python "Predicting House Price with Multiple Linear Regression.py"
```
